# Images Folder

포트폴리오 작품 이미지를 이 폴더에 추가하세요.

## 지원하는 포맷
- .jpg / .jpeg
- .png
- .webp
- .gif

## 사용 방법
1. 이 폴더에 이미지 파일을 복사하세요
2. 파일명은 자동으로 정렬됩니다 (알파벳/숫자 순서)
3. 개발 서버를 재시작하면 이미지가 갤러리에 표시됩니다

## 권장 사항
- 이미지 파일명을 의미있게 지어주세요 (예: `book-cover-forest-house.jpg`, `editorial-design-bodoni.png`)
- 이미지 크기는 최적화하여 업로드하세요 (너무 큰 파일은 로딩이 느려질 수 있습니다)
- 기본 이미지 비율은 3:2 (1920x1280)로 설정되어 있습니다

## 예시 작품
- 동화책 표지 디자인
- 에디토리얼 디자인
- 패키징 디자인
- 포스터 디자인
- 브랜딩 작업물
